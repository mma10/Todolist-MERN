Welcome to my very first full stack MERN app

-- To run the web app locally:
    
    // install server dependencies
    # npm install
    
    // install client dependencies
    # npm run client-install

    // run only the backend server (nodemon)
    # npm run server

    // run only the client
    # npm run client

    // run the app
    # npm run dev
